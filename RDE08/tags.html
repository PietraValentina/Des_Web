Tags HTML

Tag Definição Sintaxe Visualização
Table Define uma tabela. cellpadding: Distância entre a “borda” da célula até o seu conteúdo;
cellspacing: Distância entre as células da tabela.
Border: define tamanho da borda.
Del Define texto que foi deletado Texto ficará riscado no navegador.
Ins Define texto que foi inserido. Texto ficará sublinhado no navegador.
Sub Define texto substrito. Texto subscrito ficará menor na linha.
Sup Define texto subrescrito. Texto sobrescrito ficará maior na linha.
Blockquote Define uma seção citada de outra fonte. Texto ficará assim:
Os navegadores geralmente recuam os elementos.
Q Define uma citação curta. Este serve para pequenas citação, que ficrão entre aspas.
Abbr Define uma abreviatura. Fazer a utilização de título global.
Exemplo: HTML é a linguagem utilizada neste projeto.

Cite Define título a um trabalho, exemplo: obras. Texto aparecerá em italico.
A Noite Estrelada Vicent Van Gogh.
Bdo Substitui direção do texto atual. Texto ficará invertido
Code Define um espaço de codigo do computador. O texto Img em HTML define imagem.
Pre Define um texto pré-formatado. O texto 
ficará exatamente como na programação
